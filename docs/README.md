# RepoAtlas 문서 안내

`docs/` 디렉터리는 사람이 읽는 프로젝트 문서를 모아두는 공간입니다.  
이 디렉터리의 문서는 모두 한글로 작성합니다.

## 문서 원칙

- `docs/`는 사람용 문서입니다.
- 문서는 한글로 작성합니다.
- 기능 설명, 구조 설명, 운영 메모, 온보딩 문서는 이곳에 둡니다.
- AI 작업 규칙이나 에이전트 핸드오프 문서는 `.ai/`에 둡니다.

## 문서 구성

- [구조 개요](./구조-개요.md): 프로젝트 구조와 주요 디렉터리 설명
- [개발 워크플로우](./개발-워크플로우.md): 개발 및 검증 흐름
- [AI 협업 규칙](./AI-협업-규칙.md): `.ai`와 `docs`의 역할 구분
- [OpenAI 활용](./OpenAI-활용.md): 분석과 Q&A에서 OpenAI가 어디에 쓰이는지 설명

## 언제 갱신하나

- 디렉터리 구조가 바뀔 때
- 실행 방법이 바뀔 때
- 운영 방식이나 협업 방식이 바뀔 때
- 신규 개발자가 처음 볼 때 헷갈릴 수 있는 정보가 생길 때
